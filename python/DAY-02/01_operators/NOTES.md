#  Python  Operators

## 01 Arithmetic Operators:

Arithmetic operators are symbols used in Python to perform mathematical calculations on numbers.

They are one of the most basic and important parts of programming because they are used in:

- Calculations
- Logic building
- Real-world applications like banking, games, and data processing

---

### List of Arithmetic Operators in Python

Python supports **7 arithmetic operators**:

| Operator | Name | Purpose |
|----------|------|---------|
| `+` | Addition | Adds two numbers |
| `-` | Subtraction | Subtracts one number from another |
| `*` | Multiplication | Multiplies two numbers |
| `/` | Division | Gives float result |
| `%` | Modulus | Gives remainder |
| `**` | Exponentiation | Raises power |
| `//` | Floor Division | Removes decimal part |

---

## 02 Comparison Operators

### What are Comparison Operators?

Comparison operators are used in Python to **compare two values**.

They always return a result in **Boolean form**:

- `True` → condition is correct
- `False` → condition is wrong

These operators are very important in:
- Conditions (`if-else`)
- Decision making
- Loops
- Real-world logic systems

---

### List of Comparison Operators

| Operator | Name |
|----------|------|
| `==` | Equal To |
| `!=` | Not Equal To |
| `>` | Greater Than |
| `<` | Less Than |
| `>=` | Greater Than or Equal To |
| `<=` | Less Than or Equal To |

---

# 03  Logical Operators

###  What are Logical Operators?

Logical operators are used in Python to **combine multiple conditions together**.

They help in decision making when more than one condition is involved.

---

###  Logical Operators 

Python has **3 main logical operators**:

| Operator | Name | Meaning |
|----------|------|--------|
| `and` | Logical AND | All conditions must be True |
| `or` | Logical OR | At least one condition must be True |
| `not` | Logical NOT | Reverses the result |




# 04 Assignment Operators

### What are Assignment Operators?

Assignment operators are used in Python to **assign values to variables** and also to **update those values quickly**.

They make code shorter and easier to write.

---

###  Types of Assignment Operators

| Operator | Name |
|----------|------|
| `=` | Simple Assignment |
| `+=` | Add and Assign |
| `-=` | Subtract and Assign |
| `*=` | Multiply and Assign |
| `/=` | Divide and Assign |
| `//=` | Floor Divide and Assign |
| `%=` | Modulus and Assign |
| `**=` | Exponent and Assign |

---

# 05 Membership Operators

### What are Membership Operators?

Membership operators are used to **check whether a value exists inside a sequence or not**.

They return:
- `True` → value exists
- `False` → value does not exist

---

###  Sequence Types in Python

Membership operators work on:

- String
- List
- Tuple

---

###  Membership Operators

| Operator | Name |
|----------|------|
| `in` | Checks existence |
| `not in` | Checks non-existence |

---

# 06 Identity Operators

###  What are Identity Operators?

Identity operators are used to check whether two variables refer to the **same object in memory** or not.

They do NOT compare values, they compare **memory location (object identity)**.

---

###  Identity Operators in Python

| Operator | Name |
|----------|------|
| `is` | Same Object |
| `is not` | Different Object |

---

# 07 Bitwise Operators

###  What are Bitwise Operators?

Bitwise operators work on **binary numbers (0 and 1)**.

Computers understand everything in binary form, so these operators directly manipulate bits.

They are mostly used in:
- Low-level programming
- Memory optimization
- Permissions systems
- Performance-based applications

---

###  Binary Concept

- 0 = OFF
- 1 = ON


---

###  Bitwise Operators in Python

| Operator | Name |
|----------|------|
| `&` | AND |
| `|` | OR |
| `^` | XOR |
| `~` | NOT |
| `<<` | Left Shift |
| `>>` | Right Shift |
